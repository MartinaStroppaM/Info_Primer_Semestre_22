Intoducción a HTML

¿Qué es HTML?

HTML es el lenguaje con el que se define el contenido de las páginas web.

Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web , como imágenes, listas, vídeos, etc.

El HTML es un lenguaje de marcación de elementos para la creación de documentos hipertexto, muy fácil de aprender, lo que permite que cualquier persona, aunque no haya programado en la vida, pueda enfrentarse a la tarea de crear una web.

¡Armemos una lista!

  • Ul sirve para crear una lista.
  • Para cada elemento de la lista se utiliza li.
  • Section es una etiqueta semántica que me divide la info. de mi texto ("como en cajitas").
  • Body es la etiqueta que me delimita la parte que ve el usuario.
  • Main se usa para identificar la porción principal de la página.
  • Footer es una etiqueta que me da el pie de página.
  • Los iframe sirven para poner audiovisuales
  • Validator sirve para validar el código HTML.
  • Cada etiqueta tiene una tributo que se llama class. Ese atributo me permite ponerle características o estilos (es como si fuera el apellido de la etiqueta)
  • Pixel es la forma que tenemos de deterinar de forma absoluta tamaños en la pantalla
  • Margin: margen exterior
  • Pading: espacio entre la foto y el borde de la sección
